# 🌿 植境行政知識闖關訓練系統

**線上網址：** https://celestefang77.github.io/plantarium/plantarium.html

---

## 系統說明

這是一套為植境行政團隊設計的互動式學習系統，將 18 份工作懶人包轉化為闖關訓練，幫助同仁真正記住、理解並運用日常工作知識。

| 項目 | 說明 |
|------|------|
| 帳號制 | 每位同仁使用個人帳號登入，進度獨立儲存 |
| 隨機抽題 | 每關從 15 題題庫隨機抽 10 題，每次題組不同 |
| 通過標準 | 90 分 |
| 可重複挑戰 | 未通過可無限補考 |
| 成績記錄 | 自動同步至 Google Sheets，主管可查看各員工學習狀況 |

## 🧠 情境挑戰（第 19 關）

獨立的 Critical Thinking 模組，不計分。

每題都是真實工作情境，選完答案後揭示「思考框架」，訓練同仁在非預期狀況下的判斷力。

## 技術架構

```
學員端（GitHub Pages HTML）
    ↓ JSONP 登入驗證
Apps Script（Web App）
    ↓ 讀寫
Google Sheets（帳號管理 + 成績紀錄）
```

- 單一 HTML 檔案，React + Babel CDN，無需建置
- 進度存於 localStorage（以帳號為 key）
- 成績提交使用 fetch + no-cors 模式
