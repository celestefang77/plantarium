## 使用方式

請使用以下網址進入系統：

👉 [點我開始闖關](https://celestefang77.github.io/plantarium/plantarium.html)

請按照設定的帳號密碼登入。

## 系統內容

- 共 18 關・127 題
- 通過標準：90 分
- 題型：單選、多選、是非、填空
